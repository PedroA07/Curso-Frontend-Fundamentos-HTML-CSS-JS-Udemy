Principais TAGS de HTML

Aqui estão as tags mais usadas em HTML:

Títulos

Para inserirmos títulos em HTML, utilizamos a tag <h1> como no exemplo a seguir:

Entrada: <h1>Título</h1>

Saída:

Título

Mas isso vai muito além de <h1>, podemos usar os títulos com hierarquia, indo do <h1> até o <h6>como segue no exemplo a seguir:

Entrada:


    <h1>1. Título de Nível 1</h1>
    <h2>1.1 Título de Nível 2</h2>
    <h3>1.1.1 Título de Nível 3</h3>
    <h4>1.1.1.1 Título de Nível 4</h4>
    <h5>1.1.1.1.1 Título de Nível 5</h5>
    <h6>1.1.1.1.1.1 Título de Nível 6</h6>

Saída:

1. Título de Nível 1

1.1 Título de Nível 2

1.1.1 Título de Nível 3

1.1.1.1 Título de Nível 4

1.1.1.1.1 Título de Nível 5
1.1.1.1.1.1 Título de Nível 6

Parágrafos

Para inserirmos parágrafos no HTML, utilizamos a tag <p>, como segue no exeplo a seguir:

Entrada: <p>Parágrafo<p>

Saída:

Parágrafo

Quebra de linha

Para inserirmos uma quebra de linha, utilizamos a tag <br>, dentro do nosso título ou parágrafos, como segue nos exemplos a seguir:

Entrada: <p>Isso é uma <br> quebra de linha.<p>

Saída:

Isso é uma
quebra de linha.

Quebra de linha temática

Para inserirmos uma quebra de linha temática, utilizamos a tag <hr>. Ela é geralmente usada para criar uma mudança da cena de uma história, ou uma mudança de tema com uma seção.
Como segue no exemplo a seguir:

Entrada: <hr>

Saída:


Links

Em HTML é possível inserír links de conexões, tanto links internos, para chamadas dentro do HTML, quanto links externos, para chamadas de terceiros.

Os links são chamados através da tag <a>.
Dentro da tag <a> utilizaremos duas propriedades com mais frequência:

  • href="url" - Colocaremos o caminho do arquivo ou o link do site, dentro das " " onde está escrito url.

  • target="_blank" - Colocaremos a propriedade target="_blank" para abrir o link em uma nova aba.

Link Interno

Links internos são aqueles que fazem chamados à arquivos pertencentes ao projeto.

Exemplo:

Entrada: <p>Veja meu primeiro HTML, criado na <a href="Aula-01.html">Aula 01</a>!</p>

Saída:

Veja meu primeiro HTML, criado na Aula 01!

Link Externo

Links externos são aqueles que fazem chamados à arquivos ou sites que não pertencem ao projeto.

Exemplo:

Entrada: <p>Para acessar a página do Google clique <a href="http://www.google.com/">aqui</a>!</p>

Saída:

Para acessar a página do Google clique aqui!

Listas

Para inserirmos listas no HTML, utilizamos as tag <ul> para listas ordenadas e a tag <ol> para listas numeradas, e por fim a tag <li> para criar as linhas, como segue nos exemplos a seguir:

Lista Ordenada

Entrada:


    <ul>
        <li><a href="Aula-01.html">Item 1</a></li>
        <li><mark>Item 2</mark></li>
        <li>Item 3</li>    
    </ul>

Saída:

  • Item 1
  • Item 2
  • Item 3

Lista Numerada

Entrada:


    <ol>
        <li><a href="https://www.google.com/" target="_blank"">Item 1</a></li>
        <li><mark>Item 2</mark></li>
        <li>Item 3</li>    
    </ol>

Saída:

  1. Item 1
  2. Item 2
  3. Item 3
